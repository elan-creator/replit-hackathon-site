# 주최사 소개

> Replit 한국 공식 파트너, MarketFit Lab

* * *

## 🏢 MarketFit Lab (마켓핏랩)

**Replit 한국 공식 파트너**로서, 비개발자와 기업이 AI를 활용해 직접 제품을 만들 수 있도록 돕고 있습니다.

### 핵심 역할

- 🤝 **Replit 한국 공식 파트너** — Replit과 공식 파트너십 체결, 한국 시장 GTM 담당
- 🏫 **바이브코딩 워크숍 & 해커톤** — 비개발자 대상 실전 교육 프로그램 운영
- 🏗️ **기업 AI 도입 컨설팅** — Replit Enterprise 도입 지원 및 파일럿 설계
- 📝 **Replit 한국어 콘텐츠** — 공식 블로그 번역, 교육 자료 제작

* * *

## 🚀 실제 구축 사례

마켓핏랩이 Replit Agent로 직접 만들고 운영 중인 서비스입니다.

### 사례 1: 마케팅 랜딩페이지 (replit.mfitlab.com)

| 항목 | 내용 |
| --- | --- |
| **서비스** | Replit 해커톤·워크숍 신청 + 기업 도입 안내 사이트 |
| **기술 스택** | React 19, Express 5, PostgreSQL, Tailwind CSS |
| **개발 기간** | 약 2주 (비개발자가 Replit Agent로 구축) |
| **개발 비용** | **약 $89 (≈ 12만원)** |
| **월 운영비** | **약 $21~25** (Autoscale 배포 + Core 구독) |
| **주요 기능** | 반응형 디자인, 이메일 알림, 블로그 연동, 기업용 페이지 |

**포인트**: 전문 개발 에이전시에 외주하면 500~1,000만원 수준의 결과물을, 12만원에 직접 만들었습니다. **95% 이상 비용 절감.**

### 사례 2: B2B 블로그 콘텐츠 자동화 시스템 (Writer)

| 항목 | 내용 |
| --- | --- |
| **서비스** | AI 블로그 글 자동 생성 + Replit 본사 블로그 한국어 번역 |
| **기술 스택** | Python, FastAPI, Claude API, Tavily 검색 API |
| **개발 기간** | 약 2주 |
| **규모** | Python ~6,800줄, 파일 ~110개, 번역글 31개 + 생성글 27개 |
| **글 1편 생성 비용** | **약 260원** (Claude API) |
| **월 운영비** | **약 $113 (15만원)** |

**주요 기능**:
- 키워드 입력 → AI가 2~3가지 관점으로 글 자동 작성
- Tavily로 실시간 팩트체크 + 출처 인용
- SEO 자동 채점 (80점 미만 시 자동 재작성)
- Replit 본사 블로그 자동 번역 (주 2회)
- 검수함에서 사람이 최종 승인 후 발행

**포인트**: 콘텐츠 마케팅 외주 비용(월 수백만원)을 월 15만원으로 대체. 월 540편 이상 자동 생산하며, 번역 품질 관리를 위한 스타일 가이드를 자동 적용합니다.

## 🔥 해커톤 진행 원칙

### 1. 빠르게 만들고 바로 검증

```
아이디어 → 30분 안에 첫 버전 → 즉시 테스트
```

완벽한 계획보다 **동작하는 프로토타입**이 먼저입니다.

### 2. 복잡한 기능보다 동작하는 흐름 우선

```
❌ "로그인, 결제, 알림, 대시보드 다 만들자"
✅ "핵심 흐름 1개가 깔끔하게 동작하면 성공"
```

### 3. 완벽보다 배포 우선

```
❌ 디자인 완벽하게 → 기능 완벽하게 → 그러다 시간 초과
✅ 동작하는 것 먼저 배포 → 피드백 받고 개선
```
