# 참가 준비 가이드

> 해커톤 시작 전에 이것만 준비하면 됩니다

* * *

## 🎯 한 줄 정의

**해커톤 당일에 바로 시작할 수 있도록**, 계정 생성과 사전 설문만 미리 완료해 주세요.

* * *

## 왜 사전 준비가 필요한가?

4시간은 짧습니다. 당일에 계정 만들고 환경 설정하느라 30분 쓰면, 실제 빌드 시간이 크게 줄어듭니다.

| 사전 준비 없이 😰 | 사전 준비 완료 ✨ |
| --- | --- |
| 당일 계정 생성 → 30분 낭비 | 도착하면 바로 빌드 시작 |
| "뭘 만들지?" 고민하다 시간 초과 | 사전 설문에서 아이디어 확정 |
| 인터넷 느려서 설치 지연 | 설치할 것 없음 (브라우저만!) |

* * *

## 🔧 Replit이란? (3분 요약)

해커톤에서 사용할 도구를 미리 알아두면 당일이 훨씬 수월합니다.

### 한 문장 정의

**Replit = 브라우저에서 앱을 만들고, 실행하고, 배포까지 한 번에 되는 올인원 개발 플랫폼**

### 왜 Replit인가?

```
전통적 개발: 에디터 설치 → 언어 설치 → 환경설정 → 코딩 → 빌드 → 서버 → 배포 (반나절)
Replit:      로그인 → "이런 앱 만들어줘" → Deploy 클릭 → 끝 (수십 분)
```

### 핵심 기능 3가지

| 기능 | 설명 | 왜 좋은가 |
| --- | --- | --- |
| **Replit Agent** | AI에게 자연어로 설명하면 전체 앱을 생성 | 코딩 몰라도 앱 제작 가능 |
| **즉시 실행** | 코드 작성 즉시 결과 확인 (미리보기 내장) | "이게 맞나?" 바로 확인 |
| **원클릭 배포** | Deploy 버튼 하나로 인터넷에 공개 | 별도 서버 설정 불필요 |

### 비유

- **Google Docs** = 브라우저에서 문서 작성 (설치 불필요)
- **Replit** = 브라우저에서 앱 만들기 (설치 불필요)

> 💡 Google Docs가 Word를 브라우저로 가져왔듯이, Replit은 개발 환경 전체를 브라우저로 가져왔습니다.

* * *

## 📋 사전 준비 체크리스트

### Step 1: Replit 계정 생성 (3분)

1. [replit.com](https://replit.com) 접속
2. **Sign Up** 클릭
3. Google 계정 또는 이메일로 가입
4. 가입 완료 후 로그인 확인

> 💡 **Pro 계정 불필요!** 실습에 필요한 AI 토큰 비용은 운영 측에서 지원합니다. 개인 결제가 필요한 상황은 없습니다.

### Step 2: 사전 설문 제출 (5분)

운영팀에서 공유한 설문 링크에서 다음 항목을 작성해 주세요:

- **참가 목적**: 이 해커톤에서 얻고 싶은 것
- **만들고 싶은 앱**: 업무에서 자동화하고 싶은 것 1개
- **사전 숙련도**: 코딩/AI 도구 경험 수준

> 💡 **"만들고 싶은 앱"이 가장 중요합니다.** 당일 이 아이디어를 바로 빌드합니다. 막연해도 괜찮아요 — "매주 보고서 정리가 귀찮다" 수준이면 충분합니다.

**아이디어가 안 떠오르면?** 이런 것들을 생각해보세요:

| 매일 반복하는 일 | 앱으로 만들면? |
| --- | --- |
| 회의록에서 액션아이템 수작업 정리 | 텍스트 넣으면 자동 추출 |
| 고객 문의에 비슷한 답변 반복 | FAQ 기반 답변 자동 생성 |
| 엑셀에서 주간 보고서 수동 정리 | 데이터 넣으면 자동 요약 |
| 팀원 일정 취합 → 캘린더 정리 | 폼 제출 → 자동 정리 대시보드 |

### Step 3: 당일 준비물

- ✅ **노트북** (Mac/Windows 무관)
- ✅ **충전기** (4시간이니까 필수!)
- ✅ **브라우저** (Chrome 권장)
- ✅ **Wi-Fi 연결 가능 상태**

> ⚠️ **별도 소프트웨어 설치 불필요!** Replit은 브라우저에서 모든 것이 동작합니다.

## 🔧 Replit 미리 둘러보기 (선택, 10분)

시간 여유가 있다면 미리 체험해보세요. 당일 훨씬 수월합니다.

### 방법 1: Agent 모드 체험 (추천)

1. [replit.com](https://replit.com) 로그인
2. **Create Repl** 클릭
3. **Agent** 선택
4. "간단한 할 일 목록 웹앱 만들어줘" 입력
5. Agent가 앱을 만드는 과정 지켜보기

### 방법 2: 기본 Repl 체험

1. [replit.com](https://replit.com) 로그인
2. **Create Repl** 클릭
3. Template: **HTML, CSS, JS** 선택
4. **Run** 버튼 클릭 → 웹 미리보기 확인

> 💡 어느 쪽이든, **"브라우저에서 바로 결과가 보인다"**는 느낌만 잡으면 됩니다.

## ❓ 자주 묻는 질문

### "코딩을 전혀 몰라도 참가할 수 있나요?"

**네!** 이 해커톤은 비개발자를 위해 설계되었습니다. Replit Agent가 코드를 생성하고, 여러분은 **무엇을 만들지**에 집중하면 됩니다. 실제로 이전 참가자 대부분이 비개발 직군이었습니다.

### "Pro 계정이 필요한가요?"

**아니요.** 실습에 필요한 AI 토큰 비용은 운영 측에서 지원합니다. 개인 결제 요구 없습니다.

### "혼자 참가해도 되나요?"

**네.** 개인별로 각자의 앱을 만드는 방식입니다. 팀 구성이 필요 없습니다.

### "4시간 안에 정말 앱이 만들어지나요?"

**네.** 핵심은 **스코프를 작게** 잡는 것입니다. "팀 프로젝트 관리 도구"가 아니라 "회의록 액션아이템 추출기" 수준으로 좁히면 충분히 가능합니다.

### "사전에 프로그래밍 공부를 해야 하나요?"

**아니요.** 오히려 업무 도메인 지식이 더 중요합니다. "우리 팀에서 매주 반복하는 귀찮은 일"을 잘 정리해오는 게 최고의 준비입니다.

### "만든 앱을 해커톤 이후에도 계속 쓸 수 있나요?"

**네.** Replit에 배포된 앱은 계속 동작합니다. 해커톤 이후 본인 계정으로 이관하면 지속적으로 개선하며 사용할 수 있습니다.
