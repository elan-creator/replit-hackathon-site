# 사전 설문

> 해커톤 시작 전 참가자들의 아이디어와 숙련도를 파악합니다

* * *

## 🎯 한 줄 정의

**사전 설문 = 해커톤의 목표와 참가자의 기대를 맞춰주는 준비 과정**

* * *

## 왜 필요한가?

사전 설문을 통해 참가자의 아이디어를 미리 파악하여, 해커톤 당일 더 효과적인 코칭과 진행이 가능해집니다.

| 설문 없이 😰 | 설문 있으면 ✨ |
| --- | --- |
| 당일 아이디어 구상 → 시간 낭비 | 미리 아이디어 확정 → 바로 빌드 시작 |
| 참가자 수준 파악 어려움 | 맞춤형 코칭 준비 가능 |
| 강사/퍼실리테이터 준비 부족 | 예상 질문 및 케이스 미리 준비 |

* * *

## 📋 설문 양식 예시

### 1. 참가 목적

```
- 이 해커톤을 통해 얻고 싶은 것은 무엇인가요? (최대 2개)
  [ ] AI 도구 활용 능력 향상
  [ ] 업무 자동화 앱 아이디어 구체화
  [ ] 팀 내 AI 도입 가능성 탐색
  [ ] Replit 사용 경험
  [ ] 기타: __________
```

### 2. 만들고 싶은 업무형 앱 아이디어 (필수)

```
- 업무에서 어떤 부분을 AI로 자동화하고 싶으신가요? 구체적으로 적어주세요. (1개)

  예시:
  - 매주 팀 보고서를 요약해주는 앱
  - 고객 문의에 자동으로 답변해주는 챗봇
  - 회의록에서 액션아이템을 추출하는 도구

  __________________________________________________
  __________________________________________________
```

### 3. 현재 숙련도

```
- 코딩 또는 AI 도구 사용 경험이 있으신가요?
  [ ] 없음 (오늘 처음)
  [ ] 기본 개념 이해 (가끔 사용)
  [ ] 직접 코드 작성 경험 있음 (자주 사용)
  [ ] AI 앱 개발 경험 있음 (능숙하게 사용)
```

### 4. 사전 질문

```
- 해커톤에 대해 궁금한 점이 있다면 자유롭게 적어주세요.

  __________________________________________________
  __________________________________________________
```

## 💡 활용 팁

- **아이디어 구체화**: 참가자 아이디어를 바탕으로 강사가 예시를 준비할 수 있습니다.
- **수준별 코칭**: 숙련도에 따라 난이도 조절 및 맞춤형 피드백 제공이 가능합니다.
- **기대치 관리**: 참가 목적을 파악하여 해커톤 목표를 다시 한번 안내할 수 있습니다.
